# 03 - 权限管理模块

## 模块目标

实现基于角色的菜单级权限控制，区分管理员和HR角色。

---

## 权限设计

| 角色 | 权限范围 |
|------|---------|
| **ADMIN** | 全部菜单 + 系统设置 + 用户管理 + 数据同步 |
| **HR** | 业务菜单（需求、供应商、人员、工时、结算、报表） |

---

## 实现方式

### 后端
- 数据库表：sys_menu（菜单）、sys_role_menu（角色菜单关联）
- 通过 SysMenuService.getMenusByRole() 获取角色菜单

### 前端
- 路由 meta.roles 字段控制页面访问
- App.vue 根据 isAdmin 控制菜单显示
- 路由守卫 beforeEach 检查权限
